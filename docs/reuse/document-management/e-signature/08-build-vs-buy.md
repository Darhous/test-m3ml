# Build vs. Buy — E-Signature

**Documenso** preferred over OpenSign primarily on trust/maturity
signals (SOC 2 work, funded backing) despite OpenSign's cost advantage
(fully free) — for a Sensitive-Operation-adjacent Feature, this program
weights Security/Healthcare Suitability over raw cost, consistent with
every prior weighted decision. Building custom e-signature/PKI handling
is explicitly the kind of high-risk cryptographic reinvention this
program exists to avoid.
