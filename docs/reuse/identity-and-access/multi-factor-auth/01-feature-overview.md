# Feature: Multi-Factor Authentication

**Module:** Identity and Access (Platform Kernel). Covers TOTP/OTP,
WebAuthn/FIDO2, and step-up authentication for Sensitive Operations
(Constitution Section 21) — e.g., requiring re-authentication before a
`ResultVerified` or Payroll dual-control action. Same candidate pool and
Engine decision as `../authentication/` (Cross-Feature Dependency —
see `MASTER_DEPENDENCY_MATRIX.md`).
