# Risk Analysis — Multi-Factor Authentication

Inherits `../authentication/12-risk-analysis.md`. One Feature-specific
addition: over-broad step-up MFA configuration (requiring it everywhere)
would create staff friction and could produce workaround behavior
(shared devices, written-down codes) — a real usability/security
trade-off to configure deliberately (SAD-level), not a reason to avoid
MFA itself. Likelihood: Medium (configuration risk, not product risk).
Impact: Low-Medium.
