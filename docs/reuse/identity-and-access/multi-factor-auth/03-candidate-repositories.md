# Candidate Repositories — Multi-Factor Authentication

Same 4 as `../authentication/03-candidate-repositories.md`. All 4 support
TOTP; Keycloak, Authentik, and Zitadel additionally support WebAuthn/
FIDO2 (hardware key/passkey) natively. Ory requires composing Kratos's
MFA support separately (consistent with its microservices-decomposition
profile noted throughout this Module).
