# Global Search Log — BI Dashboards

**Query:** `open source BI embedded analytics Metabase vs Superset vs
Lightdash 2026 forecasting`

**Finding:** Apache Superset — ASF-governed, 63,000+ stars (context
only), production use at Airbnb/Dropbox/Lyft — is "the most complete
open source answer today" specifically for the **white-label, multi-
tenant, customer-pays-the-bill** embedded-analytics case. Metabase has
the lowest barrier to entry for internal self-service, but white-
labeling/embedding SDK depth is paywalled (Pro/Enterprise). Lightdash is
dbt-first, narrower fit. Given this platform's Confirmed White-Label
requirement (Wave 1), Superset's fit is decisive, not marginal.
