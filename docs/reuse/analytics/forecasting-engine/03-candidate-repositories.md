# Candidate Repositories — Forecasting Engine

| Candidate | License | Fit |
|---|---|---|
| Prophet (`facebook/prophet`) | MIT | Best starting fit — robust to messy/seasonal data, principled uncertainty |
| statsmodels (SARIMA) | BSD-3-Clause | Better only with clean, regular data — a stronger assumption than currently evidenced |
| NeuralProphet | MIT | Wins only on very small datasets in some benchmarks; more complex to operate |
