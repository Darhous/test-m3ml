# Final Decision — LLM Gateway / Orchestration

## Decision: **ENGINE + ADAPTER**

**Engine:** Portkey Gateway (Apache-2.0, since March 2026). Narrowly
preferred over LiteLLM for its native governance/guardrail fit with
Constitution Section 28; LiteLLM remains a credible, more-battle-tested
fallback if Portkey's shorter full-OSS track record proves a concern at
SAD time.
