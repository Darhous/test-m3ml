# Feature: Analyzer Middleware Integration

**Module:** Laboratory Execution. Bidirectional connectivity between the
platform and lab analyzers (order download, result upload) — the
"Analyzer Operations" source capability. Cross-Feature Dependency: this
is the same underlying integration-engine need as Module 4's
`hl7-integration-engine`, `astm-integration`, and
`device-protocol-adapters` (analyzers speak HL7 v2 and ASTM E1381/E1394),
not a distinct category.
