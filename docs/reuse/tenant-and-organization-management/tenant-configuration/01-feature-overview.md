# Feature: Tenant Configuration

**Module:** Tenant and Organization Management (Platform Kernel).
Per-tenant configuration values (not just on/off flags) — e.g., a
tenant's chosen notification channels, White-Label branding values, or
Egypt-specific vs. future-market localization settings. Distinct from
`feature-flags` (rollout control) though closely related — evaluated for
whether it needs a second Engine or can share the one already selected.
