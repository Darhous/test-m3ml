# Community Review — Multi-Tenancy Isolation

PostgreSQL itself is among the most widely adopted, longest-maintained
open-source database engines in existence, with governance via the
PostgreSQL Global Development Group (no single-vendor dependency) —
maximal vendor independence and long-term sustainability among every
candidate reviewed anywhere in this program so far. RLS specifically
(added in PostgreSQL 9.5, 2016) is a decade-mature feature, not
experimental.
