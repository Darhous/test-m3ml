for the exact problem (healthcare facility medical-supply inventory with
expiry/lot/cold-chain concerns), the same class of finding as OpenELIS
Global for Laboratory Execution — a healthcare-specific tool that
directly matches the domain, rather than a generic ERP module adapted to
fit.
