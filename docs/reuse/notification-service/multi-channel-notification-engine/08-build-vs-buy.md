# Build vs. Buy — Multi-Channel Notification Engine

Building multi-channel delivery (Email/SMS/Push/WhatsApp) from scratch
means integrating and maintaining N separate vendor SDKs directly —
exactly the kind of undifferentiated integration work this program exists
to eliminate. Novu wins on Architecture and self-hosting; its license
needs verification (`04-license-review.md`) before being treated as fully
settled, so this is a **Recommended, not high-confidence** Buy decision.
