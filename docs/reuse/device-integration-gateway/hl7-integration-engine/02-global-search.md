# Global Search Log — HL7 Integration Engine

**Query:** `open source HL7 v2 ASTM lab device integration engine Mirth
Connect NextGen Connect Apache Camel 2026`

**Critical finding:** Mirth Connect (NextGen Connect) — the most widely
deployed HL7 engine in US healthcare, full HL7 v2/v3/CDA/FHIR R4/X12
support — went **commercial-only starting version 4.6 (March 2025)**.
Version **4.5.2 is the last open-source (MPL 2.0) release**. This is a
major, current licensing event directly relevant to this Feature, not a
minor footnote. Apache Camel surfaced as a general-purpose alternative
("free but not healthcare-specific, requires substantial engineering
investment"). No other dedicated open-source HL7 engine of comparable
maturity was found.
