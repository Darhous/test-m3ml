# Community Review — HL7 Integration Engine

Mirth/NextGen Connect: largest HL7-engine community and hiring pool in
US healthcare per search results — but that community is now bifurcating
around the 2025 licensing change (commercial-support forks and paid-tier
migration are actively discussed in current sources). Apache Camel:
large, healthy, CNCF-adjacent general integration community, not
healthcare-specific.
